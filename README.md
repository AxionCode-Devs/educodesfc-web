## ✨ Características Pilares del Sistema

- **Control de Acceso por Roles:** Simulación de entorno multiusuario seguro con interfaces personalizadas y restricciones de permisos según el perfil activo (Administrador, Cajero y Entrenador).
- **Panel de Control Centralizado (Dashboard):** Métricas interactivas calculadas en tiempo real (miembros, eventos, ingresos mensuales, saldos pendientes) junto a una bitácora dinámica de actividad reciente.
- **Gestión Deportiva y Expedientes de Rendimiento:** Organización del club por categorías/plantillas y fichas técnicas individuales para el seguimiento estadístico avanzado de jugadores (goles, tarjetas, minutos y estado físico).
- **Directorio Inteligente y Agenda de Eventos:** Base de datos filtrable para clasificar al personal del club (Jugadores, Socios, Directivos) y un calendario coordinado con distintivos visuales de estado para partidos y entrenamientos.
- **Módulo Contable y Facturación PDF:** Control de flujos de caja estructurado por pestañas de estado (Completados, Pendientes, Atrasados). Incluye la generación de recibos corporativos listos para exportar a PDF mediante impresión nativa.
- **Motor de Reportes Optimizado para A4:** Sistema de consultas avanzadas que genera vistas previas en tiempo real y maqueta versiones de impresión física en formato A4, ajustando la orientación de página dinámicamente.
